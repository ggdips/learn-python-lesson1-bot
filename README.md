# Learn Python Lesson Telegram Bot 1

This is Python learning project

### Prerequisites
Install Python 3, pip
```
sudo apt install python3 python3-pip
```
Install modules
```
pip3 install python-telegram-bot[sock]
```
### Running
Add settings.py
```
API_KEY = "Some_TGBOT_Key"
```
Run
```
./bot.py
```
